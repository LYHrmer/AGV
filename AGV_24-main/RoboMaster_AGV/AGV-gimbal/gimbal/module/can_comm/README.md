# can_comm 多设备通信

## 简介

can_comm 模块用于多设备通信实现，主要用于控制板数据通信

## 通信协议

CAN comm的通信协议如下：

| 帧头        | 数据长度 | 数据             | 帧尾        |  crc8校验 |
| ----------- | -------- | ---------------- | -------- | ----------- |
| 's'（0x73） | 0~255    | 最大60（可修改） | 'e'（0x65） |  校验码   |
| 1-byte      | 1-byte   | n-byte           | 1-byte   | 1-byte      |
